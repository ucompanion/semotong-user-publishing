네이밍 규칙

  • 이미지 파일명과 클래스명을 동일하게 사용한다.
  • 아이콘명 : 아이콘-크기-[분류]-모양-[컬러]-[번호]
  • [분류] : 기본값 (X), 추가 가능 (예시: 한곳에 같이 사용하는 아이콘을 묶을 경우)
  • [컬러] : 기본값 (O), 생략 가능 (예시: 유튜브, 블로그 등 브랜드 로고)
  • [번호] : 기본값 (X), 추가 가능 (예시: 아이콘명 중복시 02부터 추가)
