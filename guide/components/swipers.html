Swiper

API

Default
initialSlide: 0 시작 슬라이드번호
speed: 300 속도
autoHeight: false 자동높이
roundLengths: true 텍스트가 흐리게 나오지 않도록 영역반올림
nested: true 중첩된 Swiper 이벤트차단 (부모와 방향이 같을때 사용)
Slides grid
spaceBetween: 0 간격
slidesPerView: 'auto' 슬라이드 노출개수
slidesPerGroup: 1 슬라이드 그룹개수
centeredSlides: false 모든슬라이드 센터모드
centeredSlidesBounds: false 좌우간격없는 센터모드
centerInsufficientSlides: false slidesPerView보다 슬라이드개수가 부족할때 센터모드
Touches
touchRatio: 1 터치비율
simulateTouch: true 마우스사용 허용
followFinger: true 비활성화시 손가락을 놓았을땨만 반응함
allowTouchMove: true 다음/이전 버튼으로만 슬라이드 가능
threshold: 0 슬라이드가 반응하기 시작하는 터치 거리
(Mac에서는 스와이프가 민감하여 링크 클릭시 먹통되므로 최소값을 5이상 줘야 함)
Touch Resistance
resistance: true 내성 한계를 적용한다. (슬라이드가 처음이거나 끝일때)
resistanceRatio true 내성 저항비 적용
Clicks
slideToClickedSlide: true 클릭한 슬라이드로 전환됨
Progress
watchSlidesVisibility 보이는 슬라이드에 swipe-slide-visible 클래스가 적용됨
Observer
observer 슬라이드 영역이 변경될 경우 업데이트 됨
observeParents 부모 스와이프 영역이 변경될 경우 업데이트 됨
observeSlideChildren 자식 스와이프 영역이 변경될 경우 업데이트 됨
Thumbnails
thumbs: { } swiper: 스와이퍼 이름 swiper: { el: '.swiper-container-thumbs', slidesPerView: 5, }
Methods
slideNext() mySwiper.slideNext(speed, runCallbacks); //다음 슬라이드 실행
slidePrev() mySwiper.slidePrev(speed, runCallbacks); //이전 슬라이드 실행
slideTo() mySwiper.slideTo(index, speed, runCallbacks); //지정 슬라이드 이동
updateAutoHeight() mySwiper.updateAutoHeight(speed); //자동 높이 설정
update() mySwiper.update(); //새로고침
destroy() mySwiper.destroy(deleteInstance, cleanStyles); //초기화
Events
공통예시 on: { init: function({ 실행 }); } mySwiper.on('init', function(){ 실행 });
mySwiper.init(function(){ 실행 });
slideChangeTransitionStart 에니메이션 시작시 이벤트가 다른 슬라이드로 시작됨
slideChangeTransitionEnd 에니메이션 종료후 이벤트가 다른 슬라이드로로 시작됨
resize 창 리사이즈시
observerUpdate 영역의 사이즈가 변경될 시
init 스와이프 로드시
Components
navigation: { } nextEl: '.swiper-button-next', //다음 슬라이드 prevEl: '.swiper-button-prev', //이번 슬라이드
pagination: { } 기본옵션 el: '.swiper-pagination', type: 'bullets', clickable: false,
사용자정의 1 renderBullet: function (index, className) { return '<span class="' + className + '">' + (index + 1) + '</span>'; }
사용자정의 2 renderFraction: function (currentClass, totalClass) { return '<span class="' + currentClass + '"></span>' + ' of ' + '<span class="' + totalClass + '"></span>'; }
사용자정의 3 renderCustom: function (swiper, current, total) { return current + ' of ' + total; }
Autoplay
autoplay: {} delay: 5000 //지연시간
mySwiper.autoplay.start() 자동 재생 시작
mySwiper.autoplay.stop() 자동 재생 중지
